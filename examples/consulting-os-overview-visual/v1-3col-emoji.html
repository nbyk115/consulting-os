解決する課題（典型 6 例）
  • 属人営業 / 属人提案で再現性なし、トップ営業が辞めると機能停止
  • ハルシネーション / 出典なし数値で信頼失墜、納品リスク高い
  • 提案書作成に毎回 40h、案件回転率が上がらない
  • 海外展開で日本語字形 / 翻訳 / 文化文脈の品質崩壊
  • CPC / CPA 至上主義で LTV / 粗利が見えない、PL に落ちない
  • クリエイティブ品質が制作者個人技、ブランド統一困難
解決アプローチ（佐藤裕介 + 小野寺信行）
  • 3 変数交点（誰が・何を・どう）で属人を構造化
  • 反証チェック Step 1-4（自己 / 構造 / 実用 / リスク即潰し）
  • 60/40 ルール（既存 60% / 新規 40%）で挑戦回数最大化
  • マスク式 4 原則（インセンティブ / 評判 / Unkind Truth / アルゴリズム）
  • 佐藤メソッド: 構造で売る、属人営業の推奨を排除
