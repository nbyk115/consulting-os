## 1. ご相談の整理

> 関根さんから頂いたご要望

1. 関根さんが積み上げてきたコンサル手法・営業研修トークをベースに、**関根クローン**を高精度で作る

---

## 2. なぜ NotebookLM では届かなかったのか

NotebookLM の本質：**RAG型の知識検索ツール**

| NotebookLM が得意なこと | 関根さんが求めること |
|---|---|
| 文書から関連情報を**検索・引用**する | 関根さんの**思考・判断**を再現する |
| 「何が書いてあるか」を返す | 「関根さんならどう判断するか」を返す |
| 静的な知識ベース | 動的な意思決定エンジン |

→ NotebookLM は「辞書」、関根さんが欲しいのは「**判断装置（クローン）**」。
→ そもそも設計思想が違うので、何度ソースを足しても辞書のままになる。

---

## 3. コンサルOS が解決できる理由

コンサルOS の本質：**Claude Code をベースにした、判断・思考の構造化システム**

### 3つの構成要素で「判断装置」を実現

```
┌──────────────────────────────────────────────────┐
│  関根さんver コンサルOS                              │
│                                                  │
│  ① スキル定義                                      │
│     関根さんの判断軸・思考フレーム・価値観を構造化      │
│     例：営業の場での顧客タイプ別アプローチ判断ロジック   │
│                                                  │
│  ② サブエージェント構成                              │
│     スペシャリスト思考を関根さんverで実装             │
│     例：「LP作成エージェント」「営業資料エージェント」   │
│         が関根さんの価値観でアウトプット              │
│                                                  │
│  ③ 判断ログ（ナレッジ）                             │
│     関根さんの過去判断・成功事例・失敗事例を蓄積        │
│     クローンが参照することで判断の精度が上がる         │
└──────────────────────────────────────────────────┘
```

### NotebookLM との決定的な違い

| 機能 | NotebookLM | コンサルOS（関根さんver） |
|---|:---:|:---:|
| 知識検索 | ◯ | ◯ |
| **判断フレームの再現** | ✗ | **◎** |
| **役割別エージェント** | ✗ | **◎** |
| **アウトプット品質保証** | ✗ | **◎**（バックエンドで品質チェック） |
| 継続的アップデート | △ | ◎ |
| 関根さんver化 | ✗ | ◎ |

## 5. プロジェクト（Phase 1：構築）

### 構築ステップ

| ステップ | 期間 | 内容 |
|---|---|---|
| ① インタビュー | 1ヶ月 | 関根さんの判断軸・思考フレームを抽出（5h × 4回） |
| ② 既存資料の取込み | 1ヶ月（並行） | 営業研修コンテンツ・過去提案書・判断ログ等のOS化 |
| ③ コアスキル定義 | 2週間 | 営業／組織／マネジメントの3領域でスキル設計 |
| ④ 基本テンプレ構築 | 3週間 | LP / トーク / 営業資料 / マネジメント方針 / 提案書 の5種 |
| ⑤ 試運転＆チューニング | 2週間 | 関根さんが実際の案件で試用 → 微調整 |
| ⑥ キックオフ研修 | 1日 | スペシャリスト向けに使い方研修（90分） |

**総期間：約3ヶ月**

### Phase 1 のゴール（完了定義）

- 関根さんが手を動かさなくても、スペシャリストが関根さんver OS を使って **基本5種類のアウトプットを単独で出せる** 状態
- 関根さんがアウトプットをレビューする際、「これは自分の判断軸と一致している」と感じられる精度

### Phase 1 で**含まれない**もの（Phase 2 / 別料金）

- 業界特化テンプレ（業界×3 = 1,500万円〜）
- 高度なサブエージェント構成（複数エージェントの自動連携）
- 関根さん以外のスペシャリスト向け個別カスタマイズ（深いレベル）
- N.Y.CRAFT 顧客への OS 再販・OEM 提供（→ 別契約）
