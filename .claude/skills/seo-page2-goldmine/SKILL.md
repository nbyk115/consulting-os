# seo-page2-goldmine

検索結果2ページ目に眠っている売上機会の発掘スキル。

## 原則

> **15位 → 5位の順位上昇は、新規ページを10本作るより価値がある。**

検索順位 11–20 位かつ月間100回以上表示のキーワード = "ページ2のゴールドマイン"。
タイトルタグ／H1／冒頭100語の修正だけで1ページ目に上がるケースが多い。

## 前提
- `client-context-load` 済み
- Google Search Console（GSC）データへのアクセス（直近90日）

## 実行手順

1. GSC から直近90日の検索パフォーマンスを取得
2. フィルタ: 順位 11–20 位 ＆ 表示回数 月100回以上
3. 各キーワードについて該当ページを開き、以下チェック
   - タイトルタグにキーワードが含まれるか
   - H1見出しに含まれるか
   - 冒頭100語に含まれるか
   - 文字数（薄い場合は増強対象）
   - 内部リンクの数
   - メタディスクリプション
4. **30日スプリント**を構築

```
Week 1: タイトルタグ＋H1の修正（指示ではなく確定文を提示）
Week 2: 内容が薄いページの増強（追加すべき見出し・本文を提示）
Week 3: 内部リンクの修正（リンク元ページとアンカーテキストを提示）
Week 4: メタディスクリプションの書き換え（確定文を提示）
```

## 出力フォーマット

```markdown
## ゴールドマイン キーワード一覧
| キーワード | 現順位 | 月間表示 | 該当URL | 不足要素 | 推定上昇幅 |
|---|---|---|---|---|---|

## 30日スプリント（確定文付き）
### Week 1: タイトル/H1
- [URL] : 旧 "..." → 新 "..."

### Week 2: 内容増強
- [URL] : 追加見出し: "..."

### Week 3: 内部リンク
- [リンク元URL] → [リンク先URL] : アンカー "..."

### Week 4: メタ
- [URL] : 新メタ "..."
```

## アンチパターン

- 「タイトルを最適化しましょう」と指示で終わる → 実装不能。**確定文を出す**
- 11位以下を全部対象にする → リソース分散。100表示閾値で絞る
