# hotice｜セールス資料 再構成ドラフト v0.1

代理店（電通デジタル等）向け拡販を前提に、既存12Pデッキの構成課題を補う3枚を先行作成。

## 何を直したか

既存デッキの課題：
1. 市場の"見立て"がなく、出典データの引用で止まっている
2. ROAS 20,000% / 平均600% / 50億円 等の強い数字がP5の小さな箇条書きに埋没
3. 競合（CARTA / AnyMind / UUUM / BitStar / TORIHADA）との差分が不明瞭
4. 「グローバル」が抽象語のまま
5. 代理店協業モデルの記述なし

## このドラフトに入っているスライド

| # | タイトル | 狙い |
|---|---|---|
| 01 | Proof ｜ 数字で示すhoticeの現在地 | 主要KPIを6枚の大判タイルで。稟議に使える粒度 |
| 02 | Market ｜ AI時代の見立て | 〜2023／2024-25／2026→ の3段階で主戦場の移動を描き、hoticeのTakeを提示 |
| 03 | Landscape ｜ 代理店パートナー比較 | 6社×7軸の比較表＋ポジショニングクレーム |

## 見方

`index.html` をブラウザで開くと3枚のスライド（1280×720）が縦に並びます。上部ナビから各スライドにジャンプ可能。
