## 19. 反証チェック結果

✅ Step 1: 「全 5 軸並列で進める」反論 → リソース分散、Phase 1 は軸 A + 軸 C 集中が外科的 / 「楽観 PL で投資家説得」反論 → 投資家慣れの水野さんは見抜く、誠実版（標準 -139 万）の方が長期信頼関係構築に有利

✅ Step 2: ConsultingOS 構築コスト 800 万は外注換算の試算（30 エージェント × 5-15 万 / 22 スキル × 3-10 万 等の経験則）/ Hotice 月 5 万 × 3 ヶ月受注実績は FACT、N.Y.CRAFT 案件は examples/ + strategy/ で機械検証可能 / 水野さん紹介の規模は SPECULATION 寄り

✅ Step 3: 1 ファイル PR で外科的、CLAUDE.md は触らず（115 行死守）/ 既存 strategy/ ディレクトリ構造（n-y-craft-oem-case）と同構造で標準化

🔺 残存リスク:
- ConsultingOS 構築コスト 800 万円は控えめ評価、外注実勢は 1,000-1,500 万円の可能性、ピッチでは「800 万円相当」と慎重表現
- 軸 B 紹介経由 8 社（標準）は仮説段階、Phase 1-2 で実需検証必要
- 軸 C よるのことば月 50-100 ユーザーは Phase 2 課金開始後の実測で再修正
- 36 ヶ月 4-6 倍リターンは Phase 4 以降の事業継続次第、楽観前提
- 水野さん退任後の競業避止期間（6 ヶ月-1 年）の事業範囲調整は出資契約で対応必要
- 副業規定 / 利益相反 / 名義社長スキームは Phase 1 で正式確認 + 透明開示が前提
- 紹介手数料 5-10% は商慣行レンジ、N.Y.CRAFT との交渉次第
- ペイメント審査は決済代行ごとに個別判断、法人化しても通らないリスク残存
