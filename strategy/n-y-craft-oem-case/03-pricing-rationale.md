# 価格根拠＋値引き交渉時の防御線（v3 — 関根さん戦略関係反映版）

各価格の根拠を可視化し、関根さんから値引き交渉が来た時の押し返し材料を整理する。
佐藤裕介フレーム原則：価格の根拠を自分が説明できないと、必ず値切られる。

> v3 主要変更：関根さんとの既存業務委託（月5万・壁打ち）で深度インタビュー不要 → 構築工数 110h → 約 68h に圧縮。プロジェクトフィー 100万 → 50万、Light 75万 → 30万 に引き下げ。単体時給は副業フロア割りを許容、年間トータル時給で 1.77万円（フロア超え）を担保する設計。

---

## 1. 価格設計の4軸（再掲）

副業OEMの値付けは、感覚で出すと必ず外す。以下4軸の交差点で決める：

| 軸 | 内容 |
|---|---|
| 軸1：**顧客の支払い能力** | N.Y.CRAFT がいくら払えるか（月商の5〜10%が天井） |
| 軸2：**顧客の価値** | OS が N.Y.CRAFT にもたらす経済的価値 |
| 軸3：**自分のコスト** | 構築・運用工数 × 副業時給 |
| 軸4：**代替手段の価格** | NotebookLM・他ツール・人材採用との比較 |

## 2. プロジェクトフィー：50万円（v3: 100万 → 50万）

### コストベース計算（インタビュー削減＋関根さん既存関係反映）

| ステップ | 工数 |
|---|---:|
| 関根さんヒアリング（1h × 1回／既存業務委託で関係構築済） | 1h |
| 既存資料の構造化・スキル素材化 | 8h |
| 顧客像定義書・ブランド基盤書を構築 | 8h |
| コアエージェント関根さんver化（中核15名） | 18h |
| 関根さんメソッド・スキル化（営業/組織/マネジメント） | 12h |
| 中堅・中小企業向け業界プレイブック新設 | 8h |
| 反証モード関根さん流カスタム | 4h |
| 試運転・チューニング・学習ログのセットアップ | 5h |
| キックオフ研修準備＋実施（1h × 2回） | 4h |
| 合計 | 約 68h |

副業時給 0.74万円換算：68h × 0.74万 = 50万円（戦略圧縮価格）
副業時給 1.5万円換算：68h × 1.5万 = 102万円（標準価格・参考）

### なぜ単体時給フロア割りを許容するか

理由は「単体時給」より「年間トータル時給」を見るから（佐藤裕介フレーム）：

| 観点 | 数字 |
|---|---:|
| プロジェクト単体（50万 / 68h） | 約 7,400 円/h（フロア割り） |
| 年間トータル（業務委託 60万 + プロジェクト 50万 + ランニング 180万）/ 年間工数 約 164h | 約 1.77 万円/h（フロア超え） |

「等価交換」として整理：

| 自分が支払う対価 | 関根さんから受け取る対価 |
|---|---|
| プロジェクトフィー 約 50万分の圧縮（標準102万 → 50万） | N.Y.CRAFT 名出しでの事例公開権 |
| | 構築プロセス・成果のケーススタディ化権 |
| | 他社展開時のリファレンスコール協力（年 2 回まで） |
| | 学習ログの匿名化された学習データの他案件再利用権 |
| | 関根さんネットワーク経由の他案件波及（リード源としての価値） |

→ プロジェクト単体ではなく「3 層の合算（業務委託＋プロジェクト＋ランニング）」を1 つのキャッシュフロー設計として捉える。

### 50万を切ろうとされた場合の防御線

| 関根さんの言い方 | 押し返しスクリプト |
|---|---|
| 「もう少し安くならない？」 | 「既に標準 102 万からリファレンス案件として 50 万まで圧縮しています。30 万の Light プランも用意していますが、業界プレイブック新設・顧客像/ブランド基盤書のフルカスタムが Phase 2 送りになります」 |
| 「ランニングも下げて」 | 「ランニングを下げると年間トータル時給がフロア割れし副業として成立しません。プロジェクトフィーは戦略的に圧縮していますが、ランニングは据え置きが条件です」 |
| 「最初は無料 / 後払いで」 | 「着手前一括が原則です。構築工数の大半を最初の 3 ヶ月で投下するため」 |
| 「Light プランをさらに値引きで」 | 「30 万を下回ると工数削減でも副業として割に合わなくなります。これ以下にする場合は受けない判断になります」 |

最低ライン：標準 50 万 / Light 30 万 / ランニング 月 15 万（標準）または月 12 万（Light）。
これらを下回る要求が来たら受けない（年間トータル時給がフロア割れし副業破綻）。

## 3. ランニング：月15万円（v1: 月12万 → v2: 月15万）

### コストベース計算（進化版OS反映）

| 月次タスク | 工数 |
|---|---:|
| 月次レビュー会（90分 + 準備30分） | 2h |
| 品質チェック（反証モード評価込み） | 2h |
| evolution-log メンテ（規律違反学習化） | 1.5h |
| エージェント本質レビュー（`/review-agent-essence`） | 1h |
| アップデート反映・スキル更新 | 1.5h |
| Slack/メール質問対応 | 1h |
| **合計** | **9h**（うち上限8h、超過時 +1h は時間単価2万） |

副業時給 1.7万円換算：**8h × 1.7万 + マージン = 月15万円**

### v1 から月3万増額の根拠
- evolution-log メンテと反証モード評価が追加（OS進化に伴う運用負荷）
- 月次の `/review-agent-essence` でエージェントの形骸化を継続防止
- 19スキル全体の整合性維持（関根さんメソッドの永続資産化）

### N.Y.CRAFT の支払い能力との整合

- 推定月商：80〜400万（クライアント数 × 単価から逆算）
- 月12万 = 月商の **3〜15%**
- → 月商200万以上なら 6%以下で済む。**重さを感じない比率**

### 既存業務委託 月5万との心理的距離

- 月5万 → 月12万 = **2.4倍**
- 心理的に呑める範囲（一般に3倍以内が「同じ関係性の延長」と感じられる距離）
- これを超えると「全く別物の高額案件」という認知になり、慎重姿勢になる

### 値引き交渉が来た場合の防御線

| 関根さんの言い方 | 押し返しスクリプト |
|---|---|
| 「月12万にならない？」 | 「月12万にする場合、evolution-log メンテとエージェント本質レビューを四半期1回に縮小させていただきます。OS の継続進化スピードが落ちますが大丈夫ですか？」 |
| 「v1 の月12万でいいのでは？」 | 「進化版OSは反証モード評価・evolution-log・エージェント本質レビューが組込まれており、運用工数が増えています。月12万にする場合は Light プラン（後述）に切り替えになります」 |
| 「成果が出るまで月5万でお試し」 | 「お試し期間として最初の2ヶ月のみ月10万 + 工数上限5h でいかがでしょう。3ヶ月目から通常料金に戻します」 |
| 「成果報酬だけで運用したい」 | 「成果報酬だけだとこちらのリスクが大きすぎます。最低限の運用工数を担保するベースが必要です」 |

**最低ライン：月12万 + 工数上限6h（Light プラン）**。これ以下は受けない。

## 4. 成果報酬：通常5% / 再販15%

### 通常受注 5% の根拠

**業界相場との整合**：

| ケース | 相場 | 採用 |
|---|---:|:---:|
| 紹介料 | 5〜10% | — |
| ライセンス料（売上ベース） | 3〜10% | ◯ |
| 代理店マージン | 10〜30% | — |
| プロダクトのレベニューシェア | 5〜20% | — |

**OS の貢献度評価**：
- N.Y.CRAFT のビジネスは **関根さんの営業力・関係性で売れている**
- OS は「業務効率化＋アウトプット品質向上」の補助ツール
- OS なしでも売れるが、OS があるとスケールできる
- → **OS は補助役、主役じゃない** → 5%が妥当

10% を要求しないのはなぜか：
- 副業フレーム＋関根さんとの長期関係を考えると 10% は攻めすぎ
- 関根さんが呑まない可能性が高く、契約全体が流れるリスク
- 5% でも年間数十万円は乗る → 十分な収益貢献

### 再販 15% の根拠

OS自体が商品となる場合：
- N.Y.CRAFT が他社に OS を販売／OEM 提供する
- → **OS は商品の主役そのもの**
- → 高めの料率が妥当

業界相場：プロダクトのレベニューシェアは 5〜20%、商品としての貢献度が高ければ高め。
**15% は中央値**。20% は強気すぎ、10% は控えめすぎ。

### 値引き交渉が来た場合の防御線

| 関根さんの言い方 | 押し返しスクリプト |
|---|---|
| 「成果報酬は無しで」 | 「ベース料金を上げる必要があります。プロジェクト 75 万・ランニング月 18 万にしてはいかがでしょう」 |
| 「5%は高い、3%で」 | 「3%にする場合、対象期間を24ヶ月→12ヶ月に短縮してはいかがでしょう」（実質的にこちらの取り分減らない） |
| 「再販15%は取りすぎ」 | 「再販時はこちらが OS の品質責任を負います。15%は業界平均で、これ以下にすると品質保証の根拠がなくなります」 |

**最低ライン：通常3% / 再販10%**。これ以下なら成果報酬抜きでベース料金を上げる。

## 5. 全体経済性のクロスチェック

### 自分側（標準プラン）

| 項目 | 金額 | 工数 |
|---|---:|---:|
| プロジェクトフィー | 50万 | 68h |
| ランニング | 月15万 × 12 = 180万 | 96h |
| 成果報酬（仮：N.Y.CRAFT の OS 活用売上 1,000万 × 5%） | 50万 | 5h |
| 計（業務委託除く） | 280万 / 年 | 169h / 年 |
| 業務委託（既存） | 月5万 × 12 = 60万 | （別カウント） |
| 総計 | 340万 / 年 | |

副業時給換算（業務委託除く）：280万 / 169h = 約 1.66 万円/h（フロア超え）
業務委託込み：340万 / 164h（プロジェクト 1 回分） = 約 1.77 万円/h
→ プロジェクト単体は時給 7,400 円でフロア割りだが、年間トータルでフロア超え。副業として成立ライン。

### 自分側（Light プラン）

| 項目 | 金額 | 工数 |
|---|---:|---:|
| プロジェクトフィー | 30万 | 38h |
| ランニング | 月12万 × 12 = 144万 | 72h |
| 成果報酬（仮）| 30万 | 5h |
| 業務委託 | 60万 | — |
| 総計 | 264万 / 年 | 約 115h |

副業時給換算：264万 / 115h = 約 2.30 万円/h（フロア超え）
→ Light プランは工数を大幅圧縮するため、時給ベースではむしろ標準より高い。

### N.Y.CRAFT 側

| 項目 | 標準 | Light |
|---|---:|---:|
| プロジェクトフィー | 50万 | 30万 |
| ランニング | 月15万 × 12 = 180万 | 月12万 × 12 = 144万 |
| 業務委託（既存） | 60万 | 60万 |
| 成果報酬（仮）| 50万 | 30万 |
| 合計 | 340万 / 年 | 264万 / 年 |

仮に N.Y.CRAFT 月商 200万（年商 2,400万）なら、標準で年商の 14.2%、Light で 11.0%。
仮に月商 300万（年商 3,600万）なら、標準で 9.4%、Light で 7.3%。

→ N.Y.CRAFT の実際の月商次第で、標準 / Light を打合せで提示分けする。

### Light プラン（小規模スコープ）

| 項目 | 金額 |
|---|---:|
| プロジェクトフィー | 30万円（Phase 1 スコープ縮小：業界プレイブック新設を Phase 2 送り／中核エージェント 10 名／顧客像・ブランド基盤書は既存テンプレ流用） |
| ランニング | 月12万円（学習ログのメンテ＋本質レビューは四半期1回に） |
| 成果報酬 | 通常 5% / 再販 15%（据え置き） |

## 6. 値引き交渉が来た時の総合判断ロジック

### 必ず守るライン（譲ったら副業破綻）

| 項目 | 死守ライン |
|---|---|
| プロジェクトフィー | 50万（標準）または 30万（Light プラン） |
| ランニング | 月12万 + 工数上限 6h（Light プラン）|
| 成果報酬（通常） | 3% |
| 成果報酬（再販） | 10% |
| 知財所有権 | 全てこちら帰属（譲歩ゼロ） |

### 譲歩可能なオプション（こちらの利益を守りつつ呑む形）

1. **支払いタイミング調整**：分割払い（ただし金利分上乗せ）
2. **Phase 1 のスコープ縮小**：基本テンプレ5本→3本、構築期間短縮
3. **工数上限の引き下げ**：月8h → 月6h（その分料金も下げる）
4. **成果報酬対象期間の短縮**：24ヶ月 → 18ヶ月（料率は据え置き）
5. **お試し期間の設定**：最初の1〜2ヶ月のみ料金圧縮、その後通常料金

### 撤退ラインに達した場合

死守ラインを下回る要求が出て、関根さんが折れない場合：

> 「今回はお見送りさせてください。N.Y.CRAFT の規模感とご期待される水準のバランスで、
> 私どもがご提供できる範囲とギャップがあります。
> 業務委託は現状通り続けさせていただきますので、
> またタイミングが合えばお声がけください」

→ **副業の最大の損失は「断れずに受けた合わない案件」**。
→ 業務委託の関係は維持しつつ、OEM契約だけお断りする選択肢を残しておく。

## 7. 価格の前提が崩れる場合の再評価トリガー

以下のいずれかが起きたら、価格設計を見直す：

- N.Y.CRAFT の月商が想定より大幅に上下（半分以下 or 倍以上）
- 関根さんの構築への協力度が低い（インタビュー時間が確保できない等）
- Phase 1 構築工数が80hを大幅に超える見込みになった（120h以上）
- ランニング工数が連続3ヶ月で月10h超え
- 成果報酬の対象範囲で揉め事が連続発生

→ いずれかが起きたら、契約更新時 or 緊急協議で再交渉する。
