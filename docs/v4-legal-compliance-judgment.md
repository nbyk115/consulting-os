## 2. v4 新規設計法務要素

### エンジェル出資契約形態（FACT + INFERENCE）

J-KISS を第一推奨とする（INFERENCE）。根拠:
- 1000 万円規模の個人エンジェルに種類株は設計コスト過剰（司法書士 + 弁護士 30-80 万円・INFERENCE）
- 水野氏が退任後個人活動フェーズのため議決権への関与欲求が低い可能性が高い（INFERENCE）
- 500 Startups Japan テンプレート活用で法務コスト 20-50 万円程度（INFERENCE）

選択基準マトリクス（弁護士確認必須）:

| 形態 | 採用条件 |
|---|---|
| J-KISS | 経営関与なし / 即時バリュエーション決定を回避したい場合 |
| 普通株 | 水野氏が経営関与（顧問・アドバイザリー）を希望する場合 |
| 種類株 | 追加ラウンド想定 / 希薄化防止を水野氏が要求する場合 |

バリュエーション: 1 億円キャップ候補（INFERENCE）。現時点で事業実績ゼロのため算定根拠が薄く、交渉前に弁護士 + 公認会計士確認必須。

株式譲渡制限: 定款への事前記載が必要（FACT: 会社法 107 条）。設立前であれば設立時定款に盛り込む。設立後追加は株主全員同意が必要。

個人エンジェル特有論点（INFERENCE）:
- 水野氏がジーニー現役中は利益相反 / インサイダー規制の可能性あり（退任後出資の前提と整合するが、退任タイミングの法的明確化が必要）
- 個人から会社への出資か、投資目的で設立した SPC / 合同会社経由かで税務 + 法務コスト変わる（弁護士 + 税理士確認必須）

### AGPL-3.0 残存論点

現行 00-business-plan-master.md §9-2 の分析はスコープが正しいため v4 でも維持。追加確認点:
- Twenty CRM / SuiteCRM を N.Y.CRAFT 関根案件に OEM 提供する場合、SaaS 提供該当となり Section 13 ソース開示義務が発生する（FACT: AGPL-3.0 条文）
- 回避策の優先順位: 商用ライセンス版採用 > セルフホスト支援形式（顧客環境保有）> MIT / Apache 2.0 代替ツール選定。MIT / Apache 2.0 への移行は AGPL-3.0 コンポーネントを除外 or 置換が条件（FACT）

### Anthropic Usage Policy

- ConsultingOS の顧客向け SaaS 化は Anthropic API を「再包装して提供」する形態に該当する可能性（INFERENCE）
- Anthropic Usage Policy §3（Commercial use）の確認必須。特に「サービスを通じてエンドユーザーに API 能力を提供する場合」の制約事項（弁護士 + Anthropic 利用規約の最新版確認必須、2026-05-08 現在）

## 反証チェック結果

- Step 1 自己反証: J-KISS 推奨は水野氏の意思決定スタイル未確認のバイアスを含む（INFERENCE）。水野氏が「経営に関与したい」「議決権を持ちたい」場合は普通株が優先されるため、MTG 前に意思確認必須。バリュエーション 1 億円キャップは事業実績ゼロ段階での推定値であり、水野氏側が「高すぎる」と判断するリスクがある。
- Step 2 構造反証: 水野氏の退任タイミングと出資実行タイミングが未確定。「退任後に出資」という前提が崩れた場合（在職中に出資実行）、インサイダー規制 / ジーニーとの利益相反問題が上位優先度に浮上する。03-pricing-rationale.md §2「学習データ他案件転用権」は知財帰属「全てユーザー帰属」と構造上矛盾する可能性（N&Y Craft のデータから派生した学習内容の帰属が曖昧）。
- Step 3 実用反証: 本チェックは全参照ファイル（3 件）を実読して生成（FACT）。ただし水野氏退任契約書・ジーニー NDA の実物は未入手（SPECULATION）。Anthropic Usage Policy の確認は 2026-05-08 時点の知識ベース依存（実文書の最新版は要 WebFetch 確認）。
- Step 4 残存リスク即潰し: 水野氏在職中出資リスクと 03-pricing-rationale.md §2 知財矛盾の 2 点は本セッション内で潰せない（実情報未入手）。前者は handoff-mizuno-funding-v4.md §2「出資タイミング: 退任後」の FACT を事業計画 v4 に明記し、在職中出資を実施しない旨を契約条件として盛り込む（v4 書き直し時に proposal-writer へ連携）。後者は 03-pricing-rationale.md §2 の転用権条項に「N&Y Craft の事前書面同意を条件とする」文言追加を関根さんとの契約更新時に実施（次セッション着手対象）。
