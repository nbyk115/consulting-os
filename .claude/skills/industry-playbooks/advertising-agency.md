# 広告代理店業界プレイブック

> ⚠️ **ハルシネーション禁則**: 数値・市場規模・シェア等は**断定せず、必ず一次ソースで確認**。本プレイブックは「業界構造の理解」と「参照先」を提供する。詳細は `.claude/memory/data-sources.md`。

## 1. 業界構造

### 主要プレイヤー（日本市場）
- **総合系**: 電通 / 博報堂 / ADKマーケティング・ソリューションズ / 大広 / 読売広告社
- **デジタル特化**: 電通デジタル / サイバーエージェント / セプテーニ・ホールディングス / DAC / アイレップ
- **特化型**: PR専門（プラップジャパン / オズマピーアール）/ クリエイティブブティック（GO・PARTY等）/ コンサル系（アクセンチュア・デロイトデジタル等）
- **新興デジタル**: グリー / ベクトル / メンバーズ / Sxpire 等

### 業界構造の特徴
- **ピラミッド構造**: 総合代理店 → 中堅代理店 → 制作会社 → フリーランス という多層構造
- **マージンモデル**: メディアコミッション（媒体費の 15-20% が長年標準とされていたが現在は変動）/ Fee Base / 成果報酬の混在
- **広告主直契約 vs 卸し**: メディア直契約と取次契約が並存
- **デジタル化進行**: 運用型広告は事業会社が内製化する傾向、代理店はクリエイティブ・戦略寄りへシフト
- **コンサルファームの参入**: アクセンチュア・デロイト・PwC 等がマーケティング領域に進出

## 2. 業界固有 KPI

> ⚠️ 代理店ビジネスの KPI は事業モデル（コミッション / Fee / 成果報酬）で大きく変わる。**断定的なベンチマーク数値の引用は危険**。

### Primary（事業 KPI）
- **粗利率**: 媒体費を除いた手取り（事業モデル次第で大きく変動）
- **顧客単価（年間取扱高 × マージン）**: クライアント1社あたりの年間粗利
- **クライアント継続率（更新率）**: 年次の継続案件比率
- **WIN率**: 提案コンペでの受注率
- **チームの稼働率（Utilization）**: スタッフの請求可能時間比率
- **クライアント満足度**: NPS 等

### Secondary（活動 KPI）
- **新規案件パイプライン**: 提案中・商談中の案件総額
- **既存クライアントの拡張**: アップセル・クロスセル成功率
- **メディア計測指標**: ROAS / CPA / CTR / CPC / CPM / VTR（動画完視聴率）等
- **クリエイティブ効果**: 認知リフト / 好意度 / 指名検索リフト

### 小野寺信行流の指標分岐
本プレイブックは小野寺メソッドと整合:
- **指標分岐**（ブランディング / 恒常 / ナーチャリング / 獲得で別 KPI セット）
- **広告体験ガード**（ROAS 効率化が UX 毀損していないか）
- **3S フレーム**（Show / Story / Sale × 専用 KPI）
- 詳細は CLAUDE.md「小野寺信行（電通デジタル / DII Connect）の知見」参照

## 3. 業界規制・コンプライアンス

### 必須遵守
- **景品表示法**: 表示の不当性（最上級表現・No.1 表記の根拠・優良誤認）
- **特定商取引法**: 通信販売広告の表示義務
- **薬機法**（医薬品・化粧品・健康食品の効能効果表現）
- **酒税法・たばこ事業法・道路交通法**（業界別広告規制）
- **JARO（日本広告審査機構）**: 業界自主基準
- **景品規制**: 懸賞・総付景品の限度額
- **個人情報保護法**: 広告配信での個人データ取り扱い

### デジタル広告特有
- **改正電気通信事業法（2023〜）**: 外部送信規律（Cookie / トラッキング技術の通知義務）
- **GDPR / CCPA**: 海外配信時の同意取得
- **Google Ads ポリシー / Meta 広告ポリシー**: 各プラットフォーム個別ルール
- **YouTube / TikTok 等の規制**: ステマ規制（景表法改正 2023-10）

### ステマ規制（景表法 2023-10〜）
- インフルエンサー起用時に「広告」「PR」表記必須
- 違反は景表法に基づく行政処分対象

## 4. 商習慣・契約慣行

### 契約形態
- **コミッション契約**: 媒体費の X% を代理店マージン（伝統モデル）
- **Fee 契約**: 月額固定 / プロジェクト固定（戦略・クリエイティブ寄り）
- **成果報酬**: 売上・CV ベースのコミッション（ECや D2C で増加傾向）
- **ハイブリッド**: 月額 Fee + 成果ボーナス

### 業界慣行
- **支払サイト**: 大手は 60-90日後の支払いが多い（資金繰り注意）
- **メディア前払い**: 媒体費は代理店が一旦立替え後請求（クレジットライン必要）
- **競合制限**: クライアントの同業他社案件を制限する条項あり（明示確認必須）
- **クリエイティブ著作権**: 制作物の権利帰属を契約で明確化（デフォルトはクライアント帰属が多い）

### コンペ・提案
- **提案コンペ**: 大手は3-5社に提案依頼が標準
- **オリエン**: クライアントの RFP（Request for Proposal）に基づく
- **提案資料**: ストラテジー + クリエイティブ + 予算プラン + KPI 設定
- **二次コンペ**: 大型案件では一次選抜後の二次プレゼンあり

## 5. 典型的な失敗パターン

### 失敗 1: コミッションモデルの構造的限界
- 媒体費の縮小（運用型広告の効率化）→ コミッション収入縮小
- → Fee 契約 / 成果報酬への移行、または戦略・クリエイティブ価値の再定義

### 失敗 2: クライアントの内製化に置いていかれる
- 運用型広告は事業会社の内製化が進行
- → 代理店は「内製ではできない領域」（戦略 / クリエイティブ / プラットフォーム R&D）にシフト

### 失敗 3: ステマ規制違反による行政処分
- インフルエンサー起用時の「広告」「PR」表記漏れ
- → legal-compliance-checker で全クリエイティブを事前審査

### 失敗 4: 競合制限条項の見落とし
- 既存クライアントとの競合関係を新規受注時に確認しない
- → 受注前にクライアント同業他社リストをチェック

### 失敗 5: 計測指標の自己採点（小野寺・グリーン規律）
- Walled Garden（Google / Meta）の自申告 ROAS だけで成果報告
- → MMM / Incrementality / Lift テストで独立検証（Jeff Green 規律）

## 7. 提案・分析時のチェックリスト

### 提案前
- [ ] クライアントの事業モデル（B2B / B2C / D2C / SaaS / etc.）を確認
- [ ] 既存代理店の有無・契約期間（コンペか乗り換えか）
- [ ] 競合制限条項に抵触しないか確認
- [ ] 指標分岐（ブランディング / 恒常 / ナーチャリング / 獲得）を提案で明示

### 提案中
- [ ] CPC / CPA を全施策の主指標としていないか（小野寺規律）
- [ ] Walled Garden 依存度 70% 超なら Open Internet 配分を併記（グリーン規律）
- [ ] 計測の独立性（MMM / Incrementality）を提案に含める
- [ ] ステマ規制対応（PR 表記の明示）
- [ ] 薬機法・景表法の事前チェック（legal-compliance-checker 連携）

### 提案後
- [ ] 月次レポートで指標分岐別に成果を分離報告
- [ ] 広告体験スコアの定点観測（数値追求 vs UX 毀損のバランス）
- [ ] 計測の独立検証を四半期で実施
