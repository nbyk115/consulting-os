【ConsultingOS エンジェル出資サマリ v4 / 2026 年 5 月下旬 水野氏向け】

本資料は投資推奨・投資助言ではありません。記載数値は INFERENCE（実測前モデル値）または SPECULATION（仮説試算）です。

[なぜ今か]
マーケ / アドテク業界の AI 化が加速し、運用知識がコモディティ化しています。
個別ツール（HubSpot / Marketo / Salesforce Agentforce）は乱立しますが、マーケ判断の OS は未存在。
ConsultingOS は 6 部門 27 エージェントが一気通貫で動く Claude Code ネイティブ OS として既に稼働中。
技術 × ユーザー × 市場構造の 3 変数が重なる参入窓は黎明期にあります（INFERENCE）。

[構造的優位性 / 差別化 5 軸]
・業界特化: マーケ部門 7 agent（marketing-director / performance-marketer / seo-specialist / marketing-analyst / social-media-strategist / market-researcher / pr-communications）が水野さんのジーニー領域と直接重複
・個人カスタマイズ: 27 agent に判断スタイル・知見を蓄積する設計
・評判資本起点: 現金より評判 / 知識 / 実証 ROI を真の資産とする設計思想を物理化
・反証チェック物理化: 全出力に FACT/INFERENCE/SPECULATION ラベル + Step 1-4
・OEM 提供: N&Y Craft 関根さん案件で実証中（2026-05-09 提案）

[GTM / 財務 / リスク]
Phase 0（3 ヶ月）: N&Y Craft 実証 1 件目 + 27 agent 構造化。Phase 0 末に Go/No-Go ゲート。
Phase 1（9 ヶ月）: 水野さんネットワーク経由でマーケ系 OEM 案件 2-3 件展開。
ARR シナリオ 18 ヶ月: 楽観 2,400 万 / 標準 1,000 万 / 保守 360 万円（全 SPECULATION、N&Y Craft 受注前の仮説）。
ブレイクイーブン: 標準シナリオで 12 ヶ月時点 月次 50 万円（SPECULATION）。
リスク先出し: Anthropic 依存（マルチプロバイダー化で対策）/ Sierra 侵食（垂直統合 + 蓄積が障壁）/ 個人依存（4 思想を OS に書き込み）。

[Ask: 1000 万円エンジェル出資]
出資形態: J-KISS 推奨（バリュエーションキャップ 1 億円候補 + Discount 15-20%、要協議）。普通株は経営関与希望時。
出資主体: 個人 or 法人（SPC / 合同会社）の選択は弁護士 + 税理士確認後。
出資タイミング: 退任後（在職中出資はインサイダー規制リスクのため、ジーニー退任完了を契約の前提条件とする）。
用途: マーケ部門 7 agent 拡張 350 万 / インフラ + 開発 250 万 / 法務 + 契約 150 万 / N&Y Craft 着手金 250 万。

弁護士確認待ち項目: J-KISS 契約 / 前職 NDA・退任時期 / AGPL-3.0 / Anthropic Usage Policy / 個人情報保護法 / 副業整合性。

[次のアクション 3 分岐]
1. 合意: J-KISS テンプレート締結 → 退任後着手
2. 条件付き合意: 弁護士確認 + バリュエーション再交渉 → 2-4 週間後再 MTG
3. 不成立: N&Y Craft 関根案件先行 → 別エンジェル探索
