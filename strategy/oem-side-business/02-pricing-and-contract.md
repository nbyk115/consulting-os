# 料金表＋契約条項テンプレート

引き合いが来たときに即動けるための価格表と契約条項のドラフト。
佐藤裕介フレーム原則：迷ったら高く出す、安く出して工数で回収する発想は副業を殺す。

---

## 1. 料金表

### 1.1 基本3プラン

| プラン | 初期導入 | 月額 | 工数上限 | 想定OEM先 |
|---|---:|---:|---:|---|
| Light | 300万 | 50万 | 月10時間 | 既に運用力ある先（教えなくても回せる） |
| Standard | 400万 | 70万 | 月20時間 | 標準ケース。デフォルト提示プラン |
| Heavy | 500万 | 100万 | 月30時間 | 立ち上げ期で伴走密度が必要な先 |

初期提示はStandard固定。先方が「もっと安く」と言ったらLightへ落とすのではなく「工数を減らす」交渉にする（値引きで工数据え置きは絶対しない）。

### 1.2 オプション料金

| 項目 | 単価 |
|---|---:|
| 個社カスタマイズ（テンプレ新規開発） | 50万 / 1テンプレ |
| 個社カスタマイズ（既存大幅改修） | 30万 / 1テンプレ |
| 追加研修（キックオフ以外） | 30万 / 90分 |
| 工数超過分 | 月額 ÷ 工数上限（時間単価）×1.5倍 |
| 緊急対応（48h以内応答SLA） | 月額の30%上乗せ |
| 業界独占ライセンス | 年額×3倍以上、3年前払い |

### 1.3 値引きポリシー

| 条件 | 値引き |
|---|---:|
| 年額一括前払い | 月額の1ヶ月分 |
| 2年契約コミット | 月額の2ヶ月分 |
| 3年契約コミット | 月額の3ヶ月分 |
| 紹介経由（既存OEM先からの紹介） | 初期導入のみ20%OFF |

値引きしないライン：月額の30%以上は絶対に値引かない。それ以上欲しがる相手は副業として相性が悪い。

## 2. 契約条項テンプレート（主要条項）

実際の契約書は弁護士確認のうえ作成。以下は交渉時に絶対に外さない条項のドラフト。

### 2.1 契約期間

```
本契約の有効期間は、契約締結日から1年間とする。
契約期間満了の60日前までにいずれの当事者からも書面による解約の
意思表示がない場合、本契約は同一条件にて1年間自動更新される。
```

### 2.2 工数上限・超過時の取り扱い

```
甲（OEM先）が利用できる本サービスの月次工数上限は、
別紙料金表に定めるプランに応じた時間数とする。

月次工数が上限を超過した場合、超過時間に対して以下を適用する：
(1) 月額料金 ÷ 工数上限時間 × 1.5 を時間単価として、翌月請求書に加算
(2) 3ヶ月連続で工数上限を超過した場合、両当事者は誠実に
    プラン変更（上位プランへの移行）または契約条件の見直しを協議する
```

### 2.3 提供範囲の限定

```
本サービスの提供範囲は、別紙「サービス定義書」に記載の項目に限定する。
別紙「サービス定義書」に記載のない作業（個社カスタマイズ、オペレーション
代行等）は本契約に含まれず、必要に応じて別途個別契約を締結する。
```

→ サービス定義書（`01-service-definition.md`）を契約書に別紙として添付することが重要。これがないと「常識的に含まれるよね？」攻撃で工数が膨らむ。

### 2.4 知的財産権

```
本サービスを構成するコンサルOS本体（プロンプト、スキル定義、エージェント
構成、テンプレート等を含む）の著作権その他一切の知的財産権は乙
（提供者）に帰属する。

甲は本契約期間中、甲自身の事業のために本サービスを利用する
非独占的・非譲渡的なライセンスを有する。

甲は乙の事前の書面による同意なく、本サービスを第三者に
再提供、再販売、サブライセンスすることはできない。
```

### 2.5 解約条項

```
(1) 各当事者は、契約期間満了の60日前までに書面で通知することにより、
    契約期間満了をもって本契約を解約することができる。

(2) 期間途中の解約は、原則として認めない。ただし以下の場合に限り、
    違約金（残契約期間の月額料金合計の50%）の支払いをもって
    期間途中解約することができる：
    a. 甲が事業を廃止または事業譲渡する場合
    b. 乙が本サービスの提供を停止する場合（90日前通告）
    c. 双方の合意がある場合
```

→ 副業視点での重要条項：自分（乙）から解約できる条項を必ず入れる。本業が忙しくなったら畳めるようにしておく。

### 2.6 反社条項・秘密保持・準拠法

→ 標準的な条項を使用（弁護士テンプレ流用）。詳細は契約書本体で。

## 3. 提案〜契約のフロー

```
[紹介・引き合い]
   ↓
[03-ideal-partner-profile.md でフィット判定]
   ↓ Pass
[初回ヒアリング 60分]
   ↓
[Standardプランをベースに提案書（01-service-definition.md 流用）]
   ↓
[見積提示（料金表をそのまま）]
   ↓
[契約書ドラフト（本テンプレ + サービス定義書を別紙添付）]
   ↓
[契約締結 → 初期導入 → 月次運用開始]
```

ヒアリングから契約まで、合計工数の上限を「8時間」と決めておく。
これを超えるなら、その案件は副業として割に合わない可能性が高い。

## 4. 請求・支払いサイクル

| 項目 | 条件 |
|---|---|
| 初期導入費 | 契約締結時に全額前払い（着手前） |
| 月額費用 | 当月初日締め、当月末払い（前払い基本） |
| オプション・超過分 | 翌月15日締め、翌々月末払い |
| 通貨 | 日本円（消費税別） |
| 支払方法 | 銀行振込のみ（クレカ・分割は対応しない） |

前払い原則：副業で未収リスクを背負わない。後払いを希望する相手は受けない。
